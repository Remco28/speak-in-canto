# Dictionary Data Files

Place licensed dictionary source files here for local dictionary mode.

Expected default filenames:
- `data/dictionaries/cc-cedict.u8`
- `data/dictionaries/cc-canto.u8`

You can override paths with env vars:
- `DICTIONARY_CC_CEDICT_PATH`
- `DICTIONARY_CC_CANTO_PATH`

Do not commit private/proprietary data files without checking license terms.
